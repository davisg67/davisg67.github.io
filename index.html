ABOUT ME

I have been a programmer since the late '90s. Back then I got my first opportunity as a programmer working on lab calibration software using CVI LabWindows which used an ANSI C based language. In 2004 I started working as a freelancer providing network, server and desktop support as well as business software development services with VB6/Access Database. During this time I began to realize that I much preferred programming to dealing with hardware issues.

Later, I moved on from VB6 and began developing desktop and web-based applications with Alpha5 software, a rapid application development platform and relational database management system. Alpha5 could leverage a lot, but to do more I began to learn HTML, CSS JavaScript/JQuery and SQL.

By 2015 I started working as a fullstack web developer using HTML/CSS, JavaScript and Bootstrap on the front-end and ASP.Net framework, VB.Net/Web Forms and MS SQL on the back-end. More recently I have started working with C# and learning MVC design as well as Linq and Entity framework.

I have a passion for coding, seeing my work being used and hearing how it really helped someone in their daily work. When I'm not working I enjoy PC gaming (sorry console players), reading and working on my own projects or continuing education. I enjoy anything Star Wars, Marvel or fantasy/Sci-Fi in general. I also have a really sick fascination for really bad movies and sharknados.
